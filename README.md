# 📋 Lista de Convidados

Uma aplicação desenvolvida em **Java** projetada para facilitar a criação e o gerenciamento de listas de convidados para festas, eventos e confraternizações. 

## 🚀 Funcionalidades

* **Adicionar Convidados:** Insira novos nomes à sua lista de forma rápida.
* **Remover Convidados:** Exclua nomes da lista em caso de cancelamentos.
* **Exibir Lista:** Visualize todos os convidados confirmados de maneira clara e organizada.

## 🛠️ Tecnologias Utilizadas

* **Java** (Linguagem principal)
* Orientação a Objetos (POO)
